# ListWiz

ListWiz is a ToDo list app! A user can set their username and create tasks that will appear on their homepage.

## Features

- Scroll through tasks
- Save your username
- Save Tasks to DynamoDB
- Create new tasks
- Set task status
- View task details
- View toasts
